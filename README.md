# RecordsAPI
This API service is created as a system template for entering records

## Backend
### Framework - FastAPI
Libraries such as **sqlalchemy(orm mode+async)** and **pydantic** were also used

### Tests - PyTest

## Frontend
**HTML/JS** frontend with **Tailwind CSS**

## Build and run
Clone this repository by command: `git clone https://github.com/IM4FE/RecordsAPI.git`

When you're ready, start application by running:
`docker compose up --build`.

Your application will be available at http://127.0.0.1:8000/.
